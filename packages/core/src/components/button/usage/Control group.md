Buttons can be grouped using the Control Group component.

```jsx live
<BrControlGroup direction="horizontal">
  <BrButton fullWidth={false}>
    <span>Left</span>
  </BrButton>
  <BrButton fullWidth={false}>
    <span>Center</span>
  </BrButton>
  <BrButton fullWidth={false}>
    <span>Right</span>
  </BrButton>
</BrControlGroup>
```
